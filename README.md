# SIRA Digital Website

Premium React website for SIRA Digital, built with Vite, Tailwind CSS, and Framer Motion.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173`.

## Build

```bash
npm run build
```

The production files are generated in `dist`.

## Main Files

- `src/App.jsx` contains the pages, data, sections, slider, and routing.
- `src/styles.css` contains the visual theme, responsive layout, and animations.
- `public/SiraDigitalLogo.png` contains the SIRA Digital logo.
